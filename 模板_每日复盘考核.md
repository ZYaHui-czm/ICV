## 🗓️ 第4天 <a id="day4"></a>

**主题**：条件判断 if/elif/else + 逻辑运算符

**得分**：76/100 🟡

---

### 一、名词解释（每题5分，共20分）
1. **`if`、`elif`、`else` 的执行顺序**     //作答：先执行if然后elif最后else 🟡 描述太简略——正确顺序是：从上到下依次检查，if不成立才检查elif，所有条件都不成立才执行else
2. **`and` 和 `or` 的区别**     //作答：and需要两个条件都为真，or只需要一个为真 ✅
3. **"短路逻辑"**     //作答：and的第一个条件为假时不会考虑第二个条件，or是第一个条件为真时就不再考虑第二个条件 ✅ 完全正确！
4. **Falsy 值（假值）**     //作答：falsy自带假值 🔴 太模糊了！需要具体：`0`、`""`（空字符串）、`[]`（空列表）、`None`、`False` 都是假值

### 二、代码阅读（每题10分，共20分）
```python
x = 10
if x > 5:
    print("A")
if x > 8:         # 注意这是独立if，不是elif！
    print("B")
else:
    print("C")
# 作答：AB ✅ 分析：x>5输出A；x>8为True输出B，所以else不执行
```
```python
a = 0          # Falsy
b = "hello"    # Truthy
c = []         # Falsy
if a and b:      # 0 and "hello" → False（短路）
    print("1")
elif b or c:     # "hello" or [] → True（短路）
    print("2")   # 输出2
else:
    print("3")
# 作答：2 ✅
```

### 三、代码补全（10分）

🟡 逻辑正确，但注意 `elif` 不要前面加空格，格式规范化

### 四、手写代码（30分）
**题目**：判断一个数能否同时被3和5整除
```python
num = int(input())
if num // 3 == 0 and num // 5 == 0:
    print("yes")
else:
    print("no")
```
🔴 **关键错误**：`//` 是**整除**（取商），`%` 才是**取余**（取余数）！
- `15 // 3 = 5`，`5 == 0` 为 False ❌
- `15 % 3 = 0`，`0 == 0` 为 True ✅
✅ 正确写法：`if num % 3 == 0 and num % 5 == 0:`

### 五、概念辨析（20分）
| 概念A | 概念B | 作答与点评 |
|-------|-------|-----------|
| `if x > 5 and x < 10` | `5 < x < 10` | //作答：意义等价 ✅ Python支持链式比较，两者等价 |
| `if age >= 18:` | `if age = 18:` | //作答：一个大于等于，一个等于 🟡 但严重问题没指出——`if age = 18:` 在Python里是**语法错误**！`=`是赋值不能用在条件里，会报 `SyntaxError` |

---

### ❌ 错误总结

| 错误 | 正确 | 原因 |
|------|------|------|
| `num // 3 == 0` | `num % 3 == 0` | `//`是整除取商，`%`才是取余！**15//3=5≠0**，但**15%3=0** |
| "falsy自带假值" | 0/" "/[]/None/False | 需要具体说出哪些是假值 |
| `if age = 18:` 能执行 | 会报语法错误 | `=`是赋值，不能当条件判断 |

### 🔍 理解缺口
- `//` 和 `%` 混淆——**`//` 整除**（结果是商），**`%` 取余**（结果是余数）

### 📌 核心教训
> **判断整除用 `%` 不用 `//`；等号比较 `==` 赋值单 `=`；**
> **falsy 记清楚：0空None False**
