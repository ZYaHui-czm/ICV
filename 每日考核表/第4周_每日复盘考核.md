## 🗓️ 第22天 <a id="day22"></a>

**主题**：collections — Counter + defaultdict

**学习内容**：
```python
from collections import Counter, defaultdict

# Counter — 自动计数器
words = ["apple", "banana", "apple", "orange", "banana", "apple"]
c = Counter(words)
print(c)                    # Counter({'apple': 3, 'banana': 2, 'orange': 1})

# 从字符串创建
c2 = Counter("abracadabra") # Counter({'a': 5, 'b': 2, 'r': 2, 'c': 1, 'd': 1})

# 像字典一样用
print(c["apple"])           # 3
print(c["pear"])            # 0（不存在的键返回0，不报错！）

# 最常用方法
print(c.most_common(2))     # [('apple', 3), ('banana', 2)]  前2名

# 两个 Counter 可以相加
c3 = Counter(["apple", "pear"])
print(c + c3)               # Counter({'apple': 4, 'banana': 2, 'orange': 1, 'pear': 1})

# 统计文本词频
text = "the quick brown fox jumps over the lazy dog"
word_count = Counter(text.lower().split())
print(word_count.most_common(3))  # [('the', 2), ('brown', 1), ('dog', 1)]

# defaultdict(int) — 默认值 0
dd = defaultdict(int)
dd["apple"] += 1    # 键不存在 → 自动创建，默认值0 → 0+1=1
dd["apple"] += 2    # 1+2=3
print(dd)           # defaultdict(<class 'int'>, {'apple': 3})

# defaultdict(list) — 默认值空列表
students = defaultdict(list)
students["一班"].append("小明")
students["一班"].append("小红")
print(dict(students))  # {'一班': ['小明', '小红']}

# defaultdict(set) — 防重复
tags = defaultdict(set)
tags["Python"].add("编程语言")
tags["Python"].add("编程语言")   # 重复不添加
print(dict(tags))      # {'Python': {'编程语言'}}

# 实战对比：计数
# 普通字典
count = {}
for w in words:
    count[w] = count.get(w, 0) + 1

# Counter（一行）
count = Counter(words)

# 分组对比
# 普通字典
groups = {}
for word in words:
    key = word[0]
    if key not in groups:
        groups[key] = []
    groups[key].append(word)

# defaultdict
groups = defaultdict(list)
for word in words:
    groups[word[0]].append(word)
```

**对比表**：
| | `dict` | `defaultdict` | `Counter` |
|--|--------|--------------|-----------|
| 访问不存在键 | KeyError | 自动创建默认值 | 返回 0 |
| 适用场景 | 一般键值对 | 分组/归类 | 计数/频率统计 |

**口诀**：
> Counter自动计次数，most_common看排名
> 不存在键返回0，相加相减随便算
> defaultdict带默认，int是0 list是空
> 分组归类最拿手，省去判断省代码

### ❌ 错误总结

| 错误 | 正确 | 原因 |
|------|------|------|
| defaultdict 把值求和 | `[1,3]` 等列表 | append 是追加到列表，不是 sum |
| `most_common(1)` 裸元组 | `[('l', 2)]` 列表套元组 | most_common 返回列表 |

### 📌 核心教训
> **defaultdict(list) 是往列表里追加，不是求和；most_common 返回的是列表！**
