# 📄 第1周一页式复习速查表

> 生成日期：2025年7月

---

## 1️⃣ 本周核心概念

| 概念 | 一句话定义 | 关键点 |
|------|-----------|--------|
| **变量** | 存储数据的名字标签 | `=`赋值；动态类型；变量名不加引号 |
| **字符串 str** | 不可变的文本序列 | 有引号就是字符串；`+`是拼接不是加法 |
| **f-string** | 格式化字符串 | `f"你好{name}"`；花括号放变量 |
| **索引** | 定位单个字符 | 从0开始；`s[0]`第一个；`s[-1]`最后一个 |
| **切片** | 截取一段字符 | `s[i:j]`左闭右开不含j；`s[::-1]`反转 |
| **列表 list** | 有序可变的元素集合 | `append`加一个；`extend`拆开加 |
| **for 循环** | 逐个遍历元素 | `for x in lst:`；`range(始,终,步)` |
| **if/elif/else** | 条件判断 | `==`比较，`=`赋值；冒号+缩进 |
| **逻辑运算符** | 组合条件 | `and`都真；`or`有真；`not`取反 |
| **函数** | 可重复调用的代码块 | `def f(x):`；`return`返回值（之后不执行） |
| **input()** | 获取用户输入 | 永远返回字符串；数字需`int()`转 |
| **sort()** | 原地排序 | 返回None，不要赋值接 |
| **% 取余** | 判断整除 | `15%3==0`；`//`是整除取商不是取余 |

---

## 2️⃣ 关键代码模式

```python
# 🔄 字符串反转
s = "hello"
print(s[::-1])   # "olleh"

# 📋 列表增删查改
lst = [1, 2, 3]
lst.append(4)     # [1,2,3,4]
lst.pop()         # 删尾部 → [1,2,3]
lst[0] = 99       # 改 → [99,2,3]

# 🔁 for + enumerate 带序号
for i, item in enumerate(lst, 1):
    print(f"{i}. {item}")

# 🎯 条件判断模板
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")

# 🧩 函数模板
def greet(name, greeting="你好"):
    return f"{greeting}，{name}！"

# 📥 安全输入+类型转换
num = int(input("请输入数字：").strip())
```

---

## 3️⃣ 本周最高频错误

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|------------|------|
| `"10"+"20"` 以为等于30 | 结果是 `"1020"` | 字符串`+`是拼接 |
| `s[1:4]` 以为到第5个 | 到第4个（索引1,2,3） | 切片左闭右开 |
| `nums = nums.sort()` | `nums.sort()` | sort()原地排返回None |
| `num // 3 == 0` | `num % 3 == 0` | `//`是整除取商，`%`才是取余 |
| `if age = 18:` | `if age == 18:` | `=`是赋值会报SyntaxError |
| `return yes` (没引号) | `return True` 或 `return "yes"` | 没引号被当变量→NameError |
| `lst[m:]` 切片 | `lst[i:j]` | 注意拼写！ |

> **%取余//取商，整除判断别搞混**
> **=赋值==比较，if后面加冒号**
> **def函数return回，之后代码全死掉**
> **input返回是文本，转int才能算数字**

---

## 1️⃣ 本周核心概念

| 概念 | 一句话定义 | 关键点 |
|------|-----------|--------|
| **字典** | 键值对的无序映射 | `d.get(k)`安全取值；`.items()`遍历键值对 |
| **元组** | 不可变的列表 | `(42,)`单元素加逗号；拆包`a,b=(1,2)` |
| **集合** | 不重复、无序 | `&`交`|`并`-`差`^`对称差；去重`list(set(lst))` |
| **嵌套结构** | 数据结构套数据结构 | `d[key]={}`逐层建；`report[course]`防覆盖 |
| **文件 I/O** | 程序的"硬盘" | `with open("f","r",encoding="utf-8")` |
| **异常处理** | 程序安全气囊 | `try-except-else-finally`；`finally`死活执行 |
| **列表推导式** | 一行折叠循环 | `[x*2 for x in lst if x>0]` |
| **enumerate/zip** | 带序号/多列表配对 | `enumerate(lst,1)`；`zip(a,b)`以短为准 |

## 2️⃣ 关键代码模式

```python
# 📖 词频统计
count[w] = count.get(w, 0) + 1

# 📦 拆包交换
a, b = b, a

# 🔢 集合运算
a & b  # 交集
list(set(nums))  # 去重

# 🏗️ 嵌套遍历（不覆盖）
report = {}
for course, scores in data.items():
    report[course] = {"avg": sum(scores)/len(scores)}

# 📁 文件安全读写
with open("f.txt", "r", encoding="utf-8") as f:
    for line in f:
        print(line.strip())

# ⚡ 异常处理
try:
    num = int(input())
except ValueError:
    print("不是数字")
finally:
    print("结束")

# 🧠 推导式 + zip + enumerate
[w.upper() for w in words if len(w) > 3]
{name: score for name, score in zip(names, scores)}
for i, (n, s) in enumerate(zip(names, scores), 1):
    print(f"{i}. {n}: {s}")
```

## 3️⃣ 本周最高频错误

| ❌ 错误 | ✅ 正确 | 原因 |
|--------|--------|------|
| `report = {...}` 循环覆盖 | `report[key] = {...}` | 赋值是替换不是追加 |
| `print` 代替 `return` | `return xxx` | D5老问题反复出现 |
| 函数参数不用用全局变量 | 函数内用参数名 | 形参形同虚设 |
| `readlines` 忘 `.strip()` | `line.strip()` | 末尾有`\n` |
| `"w"` 模式清空文件 | 追加用`"a"` | w会抹掉原内容 |
| `except:` 裸捕获 | `except ValueError:` | 只应捕获预期异常 |
| 文件循环内反复open | with放循环外 | 性能杀手 |
| `d[key]` 不存在崩溃 | `d.get(key, "默认值")` | KeyError |

## 4️⃣ 记忆口诀

```

with open自动关，r读w写a追加
try-except安全气囊，finally死活都上场
推导式一行收工，zip像拉链enumerate带号
文件open放循环外，一行一行往里塞
```

## 1️⃣ 本周核心概念

| 概念 | 一句话定义 | 关键点 |
|------|-----------|--------|
| **虚拟环境** | 每项目独立 Python 环境 | `python -m venv myenv`；`activate`/`deactivate` |
| **pip** | Python 包管理器 | `pip install`；`pip freeze > requirements.txt` |
| **os.path** | 跨平台路径操作 | `os.path.join()`；`os.path.exists()` |
| **json** | JSON 数据读写 | `dumps/loads` 字符串；`dump/load` 文件 |
| **datetime** | 日期时间 | `timedelta` 是时间段，纯加减 |
| **lambda** | 无名一行函数 | `lambda x: x**2`；sorted/map/filter 的 key |
| **map/filter** | 序列操作/筛选 | map 返回函数结果；filter 返回原元素 |
| **类与对象** | 蓝图与实例 | `__init__` 创建时触发；`self` 指向自己 |
| **类/实例变量** | 共享 vs 独立 | 类变量 `class` 下；实例变量 `self.xxx` |
| **继承** | 子承父业 | `class Child(Parent)`；`super().__init__()` |
| **特殊方法** | 支持内置操作 | `__str__`打印 `__eq__`比较 `__lt__`排序 |
| **装饰器** | 给函数包一层 | `@d`=`f=d(f)`；`*args **kwargs`透传 |
| **迭代器/生成器** | 现用现取省内存 | `yield` 暂停；`(...)` 生成器表达式 |

## 2️⃣ 关键代码模式

```python
# 虚拟环境 + pip
python -m venv myenv && myenv\Scripts\activate
pip install requests && pip freeze > requirements.txt

# json + datetime + sorted
json.dumps(data, ensure_ascii=False, indent=2)
now + timedelta(days=1)
sorted(lst, key=lambda x: x[1], reverse=True)

# 类 + 继承 + 特殊方法
class Student(Person):
    def __init__(self, name, age, score):
        super().__init__(name, age)
        self.score = score
    def __str__(self):
        return f"{self.name}: {self.score}"

# 装饰器
def timer(func):
    @wraps(func)
    def wrapper(*args, **kwargs):
        start = time.time()
        result = func(*args, **kwargs)
        print(f"耗时：{time.time()-start:.2f}秒")
        return result
    return wrapper

# 生成器
def chunked(lst, size):
    for i in range(0, len(lst), size):
        yield lst[i:i+size]
```

## 3️⃣ 本周最高频错误

| ❌ 错误 | ✅ 正确 |
|--------|--------|
| JSON 追加用 `"a"`+`dump` | load→append→整体 dump |
| `timedelta` 当两步运算 | 就是纯加法 |
| `map(f)` 当成 filter | map 返回函数结果 |
| `reverse=True` 当升序 | 是降序 |
| 属性名拼写错误 | 对齐定义时的名字 |
| 生成器用两次 | 一次性，用完就空 |
| `nonlocal` 不知道 | 修改外层非全局变量 |
