## 1️⃣ 本周核心概念

| 概念 | 一句话定义 | 关键点 |
|------|-----------|--------|
| **变量** | 存储数据的名字标签 | `=`赋值；动态类型 |
| **列表** | 有序、可变的元素集合 | `lst.append()` / `lst[0]` / `lst[1:3]` |
| **字典** | 键值对的无序映射 | `d.get(k)` / `d.items()` |
| **函数** | 可重复调用的代码块 | `def f(x):` / `return` |

## 2️⃣ 关键代码模式

```python
# 🔄 遍历列表
for i, item in enumerate(lst):
    print(i, item)

# 📋 列表推导式
squares = [x**2 for x in range(10) if x % 2 == 0]

# 📖 字典安全取值
value = my_dict.get("key", "默认值")

# 📁 文件读取
with open("file.txt", "r", encoding="utf-8") as f:
    content = f.read()

# ⚠️ 异常处理
try:
    result = 10 / int(x)
except ValueError:
    print("请输入数字")
except ZeroDivisionError:
    print("不能除以0")
else:
    print(f"结果是{result}")
finally:
    print("执行完毕")
```

## 3️⃣ 本周最高频错误

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|------------|------|
| `if x = 5:` | `if x == 5:` | `=`是赋值，`==`才是比较 |
| `list = [1,2,3]` | `my_list = [1,2,3]` | 不要覆盖内置函数名 |
| `for i in len(lst)` | `for i in range(len(lst))` | `len()`返回整数不能直接遍历 |
| `d[key]` 不检查 key | `d.get(key, default)` | 不存在的 key 会 KeyError |

## 1️⃣ 本周核心概念

| 概念 | 一句话定义 | 关键点 |
|------|-----------|--------|
| **变量** | 存储数据的名字标签 | `=`赋值；动态类型；变量名不加引号 |
| **字符串 str** | 不可变的文本序列 | 有引号就是字符串；`+`是拼接不是加法 |
| **f-string** | 格式化字符串 | `f"你好{name}"`；花括号放变量 |
| **索引** | 定位单个字符 | 从0开始；`s[0]`第一个；`s[-1]`最后一个 |
| **切片** | 截取一段字符 | `s[i:j]`左闭右开不含j；`s[::-1]`反转 |
| **列表 list** | 有序可变的元素集合 | `append`加一个；`extend`拆开加 |
| **for 循环** | 逐个遍历元素 | `for x in lst:`；`range(始,终,步)` |
| **if/elif/else** | 条件判断 | `==`比较，`=`赋值；冒号+缩进 |
| **逻辑运算符** | 组合条件 | `and`都真；`or`有真；`not`取反 |
| **函数** | 可重复调用的代码块 | `def f(x):`；`return`返回值（之后不执行） |
| **input()** | 获取用户输入 | 永远返回字符串；数字需`int()`转 |
| **sort()** | 原地排序 | 返回None，不要赋值接 |
| **% 取余** | 判断整除 | `15%3==0`；`//`是整除取商不是取余 |

---

## 2️⃣ 关键代码模式

```python
# 🔄 字符串反转
s = "hello"
print(s[::-1])   # "olleh"

# 📋 列表增删查改
lst = [1, 2, 3]
lst.append(4)     # [1,2,3,4]
lst.pop()         # 删尾部 → [1,2,3]
lst[0] = 99       # 改 → [99,2,3]

# 🔁 for + enumerate 带序号
for i, item in enumerate(lst, 1):
    print(f"{i}. {item}")

# 🎯 条件判断模板
if score >= 90:
    print("优秀")
elif score >= 60:
    print("及格")
else:
    print("不及格")

# 🧩 函数模板
def greet(name, greeting="你好"):
    return f"{greeting}，{name}！"

# 📥 安全输入+类型转换
num = int(input("请输入数字：").strip())
```

---

## 3️⃣ 本周最高频错误

| ❌ 错误写法 | ✅ 正确写法 | 原因 |
|------------|------------|------|
| `"10"+"20"` 以为等于30 | 结果是 `"1020"` | 字符串`+`是拼接 |
| `s[1:4]` 以为到第5个 | 到第4个（索引1,2,3） | 切片左闭右开 |
| `nums = nums.sort()` | `nums.sort()` | sort()原地排返回None |
| `num // 3 == 0` | `num % 3 == 0` | `//`是整除取商，`%`才是取余 |
| `if age = 18:` | `if age == 18:` | `=`是赋值会报SyntaxError |
| `return yes` (没引号) | `return True` 或 `return "yes"` | 没引号被当变量→NameError |
| `lst[m:]` 切片 | `lst[i:j]` | 注意拼写！ |

---

## 4️⃣ 记忆口诀

> **变量数据贴标签，字符串引号不能忘**
> **索引0起切片开右闭，s[::-1]倒着逛**
> **append加一extend拆，sort原地别赋值**
> **%取余//取商，整除判断别搞混**
> **=赋值==比较，if后面加冒号**
> **def函数return回，之后代码全死掉**
> **input返回是文本，转int才能算数字**
