## 🗂️ 项目结构

```
ICV/
├── Python_45天学习计划.md      # 📋 总纲：5等级体系、核心20%、45天日程
```

## 📊 学习等级

| 等级 | 天数 | 主题 | 目标 |
|:----:|:----:|------|------|
| **Lv.1** | D1-6 | 基础感知 | 变量/循环/条件/函数/列表 → 单文件脚本 |
| **Lv.2** | D7-14 | 结构编程 | 字典/文件/异常/推导式 → CSV数据处理 |
| **Lv.3** | D15-24 | 工具化编程 | 面向对象/标准库/pip → 命令行工具 |
| **Lv.4** | D25-34 | 工程化编程 | requests/pandas/Flask/pytest/Git |
| **Lv.5** | D35-45 | 独立项目 | 完整项目 → GitHub 部署 |

## 🚀 规则

- 每日先学后考，答案写在题目旁 (`//作答：...`)
- 考核内容全部写入 `模板_每日复盘考核.md`（单一累积文档）
- 每周末生成一页式复习速查表
- 每日完成后自动 `git push` 同步到 GitHub
